# URL Shortener

### What it does:

* URL Shortener allows users to enter a long URL and converts it into a short URL.

* The application has a URL validation to ensure only validated URLs are converted to short links 


## Technology Used

* React

* Axios

* Node.js 

* package.JSON

* gitignore

* Javascript

* ShrtCode API

* Material UI
